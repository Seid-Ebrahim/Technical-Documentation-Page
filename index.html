Introduction

Python is a widely used high-level programming language for general-purpose programming. Created by Guido van Rossum and first released in 1991, Python has a design philosophy that emphasizes code readability, and a syntax that allows programmers to express concepts in fewer lines of code.

What you should already know

This guide assumes you have the following basic background:

  • A general understanding of programming concepts.
  • Familiarity with basic data structures like lists, dictionaries, and sets.
  • Some experience with object-oriented programming.
Hello World

Let’s start by writing a simple Python program:

print("Hello, World!")
Variables

Python variables do not need explicit declaration to reserve memory space. The declaration happens automatically when you assign a value to a variable.

Here are some basic rules for defining variables:

  • A variable name must start with a letter or the underscore character.
  • A variable name cannot start with a number.
  • A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _).
  • Variable names are case-sensitive (age, Age, and AGE are three different variables).

Example:

x = 5
            y = "Hello, World!"
Data Types

In Python, variables can hold values of different types, and different operations can be performed on different types. Here are the basic data types:

  • int: Integer numbers.
  • float: Floating point numbers.
  • str: Strings, or text.
  • list: A collection of ordered, changeable items.
  • tuple: A collection of ordered, unchangeable items.
  • dict: A collection of unordered, changeable key-value pairs.
If-Else Statements

The if statement is used to test a condition. If the condition is true, a block of code is executed; if false, another block (or nothing) may be executed.

if x > 0:
                print("x is positive")
            else:
                print("x is not positive")
While Loop

A while loop allows you to execute a block of code repeatedly as long as a condition is true.

x = 0
            while x < 5:
                print(x)
                x += 1
Function Declarations

Functions are defined using the def keyword. Here’s a simple function to calculate the square of a number:

def square(n):
            return n * n
Error Handling

In Python, you can handle errors gracefully using try-except blocks. Here’s an example:

try:
                print(10 / 0)
            except ZeroDivisionError:
                print("Cannot divide by zero!")
File Operations

You can open, read, write, and close files using Python’s built-in file operations:

file = open("example.txt", "r")
            content = file.read()
            print(content)
            file.close()
Modules and Packages

Python allows you to create and import modules and packages. A module is a single Python file, and a package is a collection of modules.

import math
            print(math.sqrt(16))
OOP in Python

Python supports object-oriented programming (OOP) principles like classes, inheritance, and polymorphism. Here’s a simple class in Python:

class Dog:
                def __init__(self, name):
                    self.name = name

                def bark(self):
                    print(f"{self.name} says woof!")

            dog = Dog("Buddy")
            dog.bark()
Conclusion

Python is a versatile language with a wide range of applications, from web development to data science. This documentation is a basic introduction, but there is much more to explore. Happy coding!
